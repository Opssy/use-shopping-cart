---
title: formatCurrencyString()
---

formatCurrencyString formats the prices of the products. It takes an object shaped like this:

<table>
  <tr>
    <th>Name</th>
    <th>Type</th>
  </tr>
  <tr>
    <td>value</td>
    <td>number</td>
  </tr>
  <tr>
    <td>currency</td>
    <td>string</td>
  </tr>
  <tr>
    <td>language</td>
    <td>string</td>
  </tr>
</table>

```json5
{
  value: 400,
  currency: 'USD',
  language: 'EN', //optional
}
```

If you don't pass a value to language, it will choose language based on the browsers settings

````jsx
import { formatCurrencyString } from "use-shopping-cart/src/util"

formatCurrencyString({ value, currency })

// returns $4.00
````
