---
title: reduceItemByOne()
---

<div>

`reduceItemByOne()` removes a product from the [cartItems](cartItems) array.

```jsx
/**@jsx jsx */
import { jsx, Box, Image, Button, Flex } from 'theme-ui';
import { useShoppingCart } from 'use-shopping-cart';
import { toCurrency } from '../util';

const Product = (product) => {
  const { reduceItemByOne } = useShoppingCart();
  const { name, sku, price, image, currency } = product;
  return (
    <Flex
      sx={{
        flexDirection: 'column',
        justifyContent: 'center',
        alignItems: 'center',
      }}
    >
      <Image src={image} />
      <Box>
        <p>{name}</p>
        <p>{toCurrency({ price: price, currency })}</p>
      </Box>
      <Button
        onClick={() => reduceItemByOne(product)}
        backgroundColor={'black'}
      >
        Remove One
      </Button>
    </Flex>
  );
};
```

For displaying what's actually in the cart, refer to the `CartDisplay` component in the example:
https://github.com/dayhaysoos/use-shopping-cart/blob/master/example/src/components/cart-display.js
