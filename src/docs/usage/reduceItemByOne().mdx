---
title: reduceItemByOne()
---

<div>

`reduceItemByOne(sku)` reduces the number of a product, by its sku, in the user's cart by one. This allows the user to reduce the quantity of an item in the cart by one at a time if they added too many.

```jsx
import { useShoppingCart, formatCurrencyString } from 'use-shopping-cart'

function Product(product) {
  const { reduceItemByOne } = useShoppingCart()
  const { name, sku, price, image, currency } = product
  return (
    <div
      style={{
        display: 'flex',
        flexDirection: 'column',
        justifyContent: 'center',
        alignItems: 'center',
      }}
    >
      <img alt={'Alt Text'} src={image} />
      <div>
        <p>{name}</p>
        <p>{formatCurrencyString({ price: price, currency })}</p>
      </div>
      <button onClick={() => reduceItemByOne(sku)}>Remove One</button>
    </div>
  )
}
```

For displaying what's actually in the cart, refer to the `CartDisplay` component in the example:
https://github.com/dayhaysoos/use-shopping-cart/blob/master/example/src/components/cart-display.js
