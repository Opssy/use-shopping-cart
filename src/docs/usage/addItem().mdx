---
title: addItem()
---

<div>

`addItem()` adds product objects to the [cartItems](api) array. Product objects require a
name, sku, price and currency values as shown below.

```json
{
  "name": "Bananas",
  "sku": "sku_GBJ2Ep8246qeeT",
  "price": 400,
  "image": "https://www.fillmurray.com/300/300",
  "currency": "USD"
}
```

Notice that we spread the product object when we pass it to addItem(). This is so that you are free to
add other values for your own needs.

```jsx
/**@jsx jsx */
import { jsx, Box, Image, Button, Flex } from 'theme-ui';
import { useShoppingCart } from 'use-shopping-cart';
import { toCurrency } from '../util';

const Product = (product) => {
  const { addItem } = useShoppingCart();
  const { name, sku, price, image, currency } = product;
  return (
    <Flex
      sx={{
        flexDirection: 'column',
        justifyContent: 'center',
        alignItems: 'center',
      }}
    >
      <Image src={image} />
      <Box>
        <p>{name}</p>
        <p>{toCurrency({ price: price, currency })}</p>
      </Box>
      <Button onClick={() => addItem({ ...product })} backgroundColor={'black'}>
        Add To Cart
      </Button>
    </Flex>
  );
};
```

For displaying what's actually in the cart, refer to the `CartDisplay` component in the example:
https://github.com/dayhaysoos/use-shopping-cart/blob/master/example/src/components/cart-display.js
