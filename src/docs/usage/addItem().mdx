title: addItem()
---

<div>

`addItem()` adds product objects to the [cartItems](api) array. Product objects require a
name, sku, price, and currency values as shown below.

```json
{
  "name": "Bananas",
  "sku": "sku_GBJ2Ep8246qeeT",
  "price": 400,
  "image": "https://www.fillmurray.com/300/300",
  "currency": "USD"
}
```

You can add anything you'd like to the product object.

```jsx
import { useShoppingCart, toCurrency } from 'use-shopping-cart';

function Product(product) {
  const { addItem } = useShoppingCart();
  const { name, sku, price, image, currency } = product;
  return (
    <div
      style={{
        display: 'flex',
        flexDirection: 'column',
        justifyContent: 'center',
        alignItems: 'center',
      }}
    >
      <img alt={'Alt Text'} src={image} />
      <div>
        <p>{name}</p>
        <p>{toCurrency({ price: price, currency })}</p>
      </div>
      <button onClick={() => addItem({ ...product })}>Add To Cart</button>
    </div>
  );
}
```

For displaying what's actually in the cart, refer to the `CartDisplay` component in the example:
https://github.com/dayhaysoos/use-shopping-cart/blob/master/example/src/components/cart-display.js
